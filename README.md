# 🤖 AI Chat Assistant

## AI-Powered Question Answering Assistant

### 📖 Overview

AI Chat Assistant is a web-based Generative AI application developed during an AI internship. The application enables users to ask questions in natural language and receive AI-generated responses using the Groq API and the Llama 3.3 language model.

The project demonstrates the integration of a FastAPI backend with a responsive frontend to deliver real-time AI-powered conversations.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Akshaya-Biju/AI-Chat-Assistant.git
cd AI-Chat-Assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key
```

Run the application:

```bash
uvicorn app:app --reload
```

Open **index.html** in your browser and start interacting with the AI assistant.
